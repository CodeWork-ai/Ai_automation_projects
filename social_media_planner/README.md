# Social Media Content Planner with Gemini AI

AI-powered social media content generator using Google's Gemini AI

## Setup
1. Get Gemini API key from: https://ai.google.dev/
2. Install dependencies: `pip install -r requirements.txt`
3. Add your API key to `.env` file: `GEMINI_API_KEY=your_key_here`
4. Run: `python main.py`

## Features
- Weekly content planning (1-4 weeks)
- Platform-specific optimization (Instagram, Twitter, LinkedIn, Facebook)
- AI-generated post ideas, captions, and hashtags
- Export to JSON format

## Usage
Enter your brand info and campaign goals to get professional content plans!
